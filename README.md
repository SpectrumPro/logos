# Spectrum Logos

This repository contains the official **Spectrum** logo assets. This exists as a centralized, version-controlled source of truth for all Spectrum branding assets.

## What’s Included

- Master logo designs in **SVG** format  
- Light and dark logo variants  
- Pre-exported **PNG** icons in common resolutions  
- Platform-specific application icons (.ico for Windows, .icns for macOS)  
- A script to regenerate all scaled PNG imaged from an SVG file
